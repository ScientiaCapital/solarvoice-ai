# SolarVoice AI - Voice Commands Reference Guide

## Overview

SolarVoice AI understands natural language, so you can speak to it like you would to a colleague. This guide provides examples of voice commands organized by category to help you get the most out of the platform.

## Quick Start Commands

### Getting Help

- "Show me how to report a safety issue"
- "What voice commands are available?"

### System Status
- "What's my current project status?"
- "Show me today's schedule"
- "Are there any urgent alerts?"

## Pro Tips for Effective Voice Commands

### 1. Be Clear and Specific
- ‚ùå "Need help"
- ‚úÖ "Need electrician at inverter station 3"

### 2. Include Location When Relevant
- ‚ùå "Safety issue"
- ‚úÖ "Safety issue at Zone B panel installation"

### 3. Use Natural Language
- The AI understands context, so speak naturally
- No need to memorize exact phrases

### 4. Confirm Critical Actions
- For safety-critical commands, the system will ask for confirmation
- Simply say "Yes" or "Confirm" to proceed

### 5. Hands-Free Operation
- Start commands with "Hey SolarVoice" for hands-free activation
- Useful when wearing gloves or carrying equipment

## Troubleshooting Voice Commands

### Command Not Recognized?
1. Speak clearly and at normal pace
2. Reduce background noise if possible
3. Try rephrasing with more detail
4. Check your microphone connection

### Getting Wrong Response?
1. Be more specific about what you need
2. Include relevant context (location, equipment type, etc.)
3. Use the "Help" command to see examples

### System Not Responding?
1. Check internet connectivity
2. Verify your device is logged in
3. Try the backup offline commands
4. Contact support if issues persist
