## Solar-Specific Voice Commands

### Installation Commands

#### Panel Management
- **"Starting panel installation in [location]"**
  - Example: "Starting panel installation in Zone A, row 5"
  - Logs GPS coordinates and timestamp

- **"Panel serial number [number]"**
  - Example: "Panel serial number ABC123XYZ installed position 5-12"
  - Creates digital record for warranty tracking

- **"Panel defect found"**
  - Triggers quality control protocol
  - Photos can be attached via app
  - Replacement automatically ordered

#### String Configuration
- **"String [number] connected"**
  - Example: "String 7 connected to combiner box 2"
  - Updates electrical diagram

- **"DC voltage reading [value]"**
  - Example: "DC voltage reading 450 volts on string 3"
  - Logs for commissioning reports

### Production Tracking
- **"[Number] panels installed today"**
  - Example: "186 panels installed today"
  - Calculates installation rate
  - Projects completion timeline

- **"Current installation rate"**
  - Provides real-time productivity metrics
  - Compares to project targets
