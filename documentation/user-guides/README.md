### Most Used Voice Commands
1. **"Good morning"** - Check in for the day
2. **"What's my assignment?"** - Get your tasks
3. **"Safety issue at [location]"** - Report hazards
4. **"Need [equipment]"** - Request resources
5. **"Progress update"** - Log completed work

- **Support Hotline**: 1-800-SOLAR-AI

- Speak naturally - no need for robot talk
- Include location details for better response
- Use offline mode in poor signal areas
- Practice commands during downtime
