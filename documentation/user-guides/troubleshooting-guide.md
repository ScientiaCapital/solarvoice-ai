## Voice Command Issues

### Problem: SolarVoice doesn't understand my commands

#### Quick Fixes:
1. **Check microphone**
   - Ensure microphone isn't blocked by case or debris
   - Clean microphone opening with soft brush
   - Remove any protective films

2. **Improve recognition**
   - Speak slower and more clearly
   - Face away from loud equipment
   - Cup hand around microphone in noisy areas

3. **Retrain voice profile**
   - Go to Settings > Voice > Retrain Voice
   - Complete training in quiet environment
   - Use your normal speaking voice

#### Advanced Solutions:
- Update app to latest version
- Clear app cache: Settings > Apps > SolarVoice > Clear Cache
- Check language settings match your spoken language
- Enable "Construction Site Mode" for better noise handling

### Problem: Voice commands work but get wrong response

#### Solutions:
1. **Be more specific**
   - Include location: "Zone A" instead of just "here"
   - Include quantity: "5 panels" instead of "panels"
   - Include timing: "this afternoon" instead of "later"

2. **Check context settings**
   - Verify correct project is selected
   - Confirm your role is set correctly
   - Update your current location/zone

3. **Use alternative phrasing**
   - "I need help" → "Request assistance at Zone B"
   - "Equipment problem" → "Forklift 3 has hydraulic leak"

### Problem: Voice activation not working ("Hey SolarVoice")

#### Solutions:
1. Enable voice activation: Settings > Voice > Voice Activation
2. Increase sensitivity: Settings > Voice > Activation Sensitivity
3. Train wake word: Settings > Voice > Train Wake Word
4. Check battery saver isn't disabling features

## Login Issues

### Problem: Can't log in with phone number

#### Solutions:
1. **Verify phone number format**
   - Include country code (+1 for US)
   - No spaces or dashes
   - Example: +12135551234

2. **Check verification code**
   - Code expires in 5 minutes
   - Check SMS spam folder
   - Request new code if needed

3. **Account issues**
   - Verify account is active with supervisor
   - Check if phone number was changed
   - Contact support for account recovery

### Problem: Forgot team code

#### Solutions:
1. Ask your supervisor or crew chief
2. Check welcome email/text
3. Look for posted codes at job site
4. Contact support with company name

### Problem: Account locked

#### Reasons and solutions:
- Too many failed attempts: Wait 30 minutes
- Security concern: Contact support immediately
- Expired account: Check with supervisor
- Payment issue: Company admin must resolve

## App Performance Issues

### Problem: App crashes frequently

#### Immediate fixes:
1. **Force close and restart**
   - Android: Recent apps > Swipe away SolarVoice > Reopen
   - iPhone: Swipe up > Swipe away SolarVoice > Reopen

2. **Clear app data**
   - Settings > Apps > SolarVoice > Clear Cache
   - Note: This won't delete your account

3. **Free up phone storage**
   - Delete unused apps
   - Clear photo/video storage
   - Need at least 500MB free space

#### Long-term solutions:
- Update to latest app version
- Update phone operating system
- Uninstall and reinstall app
- Check for conflicting apps

### Problem: App running slowly

#### Solutions:
1. **Reduce background apps**
   - Close unused apps
   - Disable auto-start apps
   - Restart phone weekly

2. **Optimize SolarVoice settings**
   - Disable video features if not needed
   - Use "Low Data Mode"
   - Turn off animation effects

3. **Phone maintenance**
   - Clear system cache
   - Check for malware
   - Consider factory reset (backup first!)

### Problem: Features missing or not working

#### Check these:
1. App version is current
2. Account permissions are correct
3. Feature is enabled for your role
4. Company subscription includes feature
5. Not in restricted/offline mode

## Connectivity Issues

### Problem: No internet connection

#### Offline features still available:
- Emergency commands
- Basic safety reporting
- Check-in/out (syncs later)
- View cached information

#### Solutions for connection:
1. **Check basics**
   - Toggle airplane mode on/off
   - Restart phone
   - Check cellular data is enabled
   - Verify data plan isn't exhausted

2. **Improve signal**
   - Move to open area
   - Get closer to site WiFi
   - Use signal booster if available
   - Switch between WiFi/cellular

3. **WiFi issues**
   - Forget and rejoin network
   - Check password is correct
   - Verify MAC address isn't blocked
   - Contact site IT support

### Problem: Slow sync or uploads

#### Solutions:
1. Schedule syncs for off-peak times
2. Use WiFi instead of cellular
3. Reduce photo quality in settings
4. Clear pending upload queue
5. Check for network congestion

## Battery Issues

### Problem: Battery draining too quickly

#### Power-saving tips:
1. **Optimize location services**
   - Set to "While Using App"
   - Disable high accuracy mode
   - Turn off in safe zones

2. **Adjust app settings**
   - Enable Low Power Mode
   - Reduce screen brightness
   - Disable video features
   - Turn off vibration feedback

3. **Voice settings**
   - Disable always-on listening
   - Use push-to-talk mode
   - Reduce voice sensitivity

4. **General phone tips**
   - Close background apps
   - Disable unnecessary notifications
   - Use battery saver mode
   - Keep phone cool

### Problem: Phone dying during work day

#### Solutions:
1. Carry portable battery pack
2. Use vehicle chargers during transit
3. Charge during breaks
4. Keep spare battery if removable
5. Use tablet at base station

## Safety System Issues

### ⚠️ Problem: Emergency features not working

**CRITICAL: If safety features fail, use traditional methods immediately**

#### Immediate actions:
1. Use physical emergency buttons
2. Call 911 directly
3. Use radio/phone for help
4. Alert supervisor immediately

#### Troubleshooting:
1. Check location services are enabled
2. Verify emergency contacts are set
3. Ensure notifications are enabled
4. Test in safe environment
5. Report issue immediately

### Problem: Safety alerts not receiving

#### Solutions:
1. **Check notification settings**
   - Phone: Settings > Notifications > SolarVoice > All ON
   - App: Settings > Safety > All Alerts ON

2. **Verify alert zones**
   - Confirm work location is correct
   - Check alert radius settings
   - Verify not in "Do Not Disturb"

3. **Test alert system**
   - Use "Test Safety Alert" feature
   - Confirm others receiving alerts
   - Check with safety officer

## Location Services Issues

### Problem: Location not updating

#### Solutions:
1. **Enable location services**
   - Phone: Settings > Privacy > Location > SolarVoice > Always
   - App: Settings > Permissions > Location > High Accuracy

2. **Improve GPS signal**
   - Move to open area
   - Wait for GPS lock (can take 30 seconds)
   - Restart location services
   - Calibrate compass

3. **Manual location update**
   - Use "Set Location Manually" option
   - Select zone from list
   - Verify with supervisor

### Problem: Wrong location showing

#### Fixes:
1. Calibrate GPS: Settings > Location > Calibrate
2. Clear location cache
3. Check time zone is correct
4. Disable VPN if using
5. Update zone boundaries with admin
