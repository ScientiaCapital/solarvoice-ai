# Static Assets

This directory contains static assets for the SolarVoice application.

## Required Files:

- `favicon.ico` - Main favicon
- `icon-192x192.png` - PWA icon (192x192)
- `icon-384x384.png` - PWA icon (384x384)
- `icon-512x512.png` - PWA icon (512x512)
- `apple-touch-icon.png` - Apple touch icon (180x180)
- `og-image.png` - Open Graph image (1200x630)
- `twitter-image.png` - Twitter card image (1200x600)

## Generating Icons

You can use online tools or scripts to generate these icons from a single source image:

1. **Favicon Generator**: https://favicon.io/
2. **PWA Asset Generator**: https://pwa-asset-generator.com/
3. **Real Favicon Generator**: https://realfavicongenerator.net/

## Image Optimization

All images should be optimized for web:

- Use WebP format where possible
- Compress PNG files
- Use appropriate dimensions
- Consider using Next.js Image component for automatic optimization
